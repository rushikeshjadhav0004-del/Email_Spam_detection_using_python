This project focuses on building a **Spam Email Detection System** using **Machine Learning** techniques.  
It classifies emails or text messages as **spam** or **ham (not spam)** using the **Naive Bayes algorithm** and **TF-IDF vectorization** for text feature extraction.

The model has been trained and evaluated on the **SMS Spam Collection Dataset**, achieving high accuracy in detecting spam messages.

---

##  Features
- Preprocessing and cleaning of text data  
- Text vectorization using **TF-IDF (Term Frequency - Inverse Document Frequency)**  
- Model training using **Multinomial Naive Bayes**  
- Evaluation using accuracy, precision, recall, and F1-score  
- Real-time spam message detection  
- Confusion matrix visualization
